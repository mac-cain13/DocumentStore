ExpressionComparisonOperator

public enum ExpressionComparisonOperator

Defines how to compare two Expressions.

  • lessThan: <
  • lessThanOrEqualTo: <=
  • greaterThan: >
  • greaterThanOrEqualTo: >=
  • equalTo: ==
  • notEqualTo: !=
  • like: Matches a pattern on a string, similar in behavior to SQL LIKE.
