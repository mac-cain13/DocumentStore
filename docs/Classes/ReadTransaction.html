ReadTransaction

public class ReadTransaction : ReadableTransaction

A transaction on a DocumentStore that only can perform reads, no writes are possible.

  • count(matching:)

    Count the number of Documents matching the given Query

    Precondition

    The DocumentDescriptor of the Documents you are counting must be registered with the DocumentStore the given ReadTransaction is associated with. If this isn’t the case a TransactionError.documentStoreError is thrown the DocumentStoreError will be of kind documentDescriptionNotRegistered.

    Throws

    TransactionError on all failures

    Declaration

    Swift

    public func count<DocumentType>(matching query: Query<DocumentType>) throws -> Int where DocumentType : Document

    Parameters

    query

    The Query to match the documents against

    Return Value
